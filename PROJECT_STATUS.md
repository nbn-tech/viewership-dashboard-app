# 視聴率ダッシュボード 開発状況

## アプリのゴール

名古屋地区在名7局の視聴率を分析・閲覧できる社内ダッシュボード。

最終的には **フロントエンドはAWSからデータを受け取って表示するだけ** の構成にする。
毎日データが蓄積され、年をまたいで過去を振り返れるシステム。
開発完了後はデモデータはゼロで、全て実データをもとにしたサイトになる。

### 対象データ
- 在名7局（NBN・THK・CTV・CBC・NHK・NHKE・TVA）
- 名古屋地区・世帯視聴率（1分刻み）
- 朝帯（5:30〜8:30）・夕方帯（16:00〜19:30）

---

## 現在の構成（デモ段階）

| データ種別 | 現状 | 保存場所 |
|-----------|------|---------|
| 視聴率 | 4/01〜4/14・4/17のみ実データ。それ以外はシード乱数のデモデータ | `src/data/ratings_data.json`（JSバンドル） |
| 番組表 | ✅ 完成。2025-06-25〜直近月曜、毎週月曜に1週間分更新 | S3 CSV（`bangumi-info` バケット） |
| 動画 | 4/17ドデスカのみ対応 | S3（`dodesca-video` バケット） |
| コーナー分類 | 4/17ドデスカのみ実データ、他はダミー | JSにハードコード |
| 番組内容・コーナー情報 | ほぼダミーデータ | JSにハードコード |
| 天気 | ✅ 完成。Open-Meteo → DynamoDB → API Gateway → フロント | DynamoDB `weather_daily` |

### 視聴率データの仕組み（現状）
```
src/data/視聴率_午前・午後/*.xlsx（実測値）
  → ratings_data.json に変換してJSバンドル
  → genRatings(date, slot) で日付・時間帯ごとに取得
     ├── 実データあり → ratings_data.json を使用
     └── 実データなし → シード乱数でデモデータを生成
```

## 機能別 進捗

### ✅ 完成

#### 番組表〜番組別〜
- S3 CSV取得・表示
- 選択可能期間: 2025-06-25〜直近月曜（範囲外はカレンダーでグレーアウト）
- 番組クリックでDashboardに遷移（局・日付・時刻・ズームが連動）

#### Dashboard
- 視聴率折れ線グラフ（朝/夕方切り替え、ズーム・パン）
- セグメントタイムライン
- 番組表〜コーナー別〜モード
- 日付セレクタ横の天気バッジ
- 4/17ドデスカ動画再生（グラフクリックでシーク連動）

#### Search
- キーワード検索・AI意味検索
- 検索結果モーダル（◀▶で検索結果順ナビゲーション）
- 日付列・モーダルへの天気表示

#### Analysis
- Daily/Weekly AI分析テキスト生成
- トピック分析
- 日付セレクタ横の天気バッジ

#### AWS（天気）
- DynamoDB テーブル `weather_daily`（パーティションキー: `date`、オンデマンド）
- Lambda `weather_batch_initial`（過去分一括投入、初回のみ実行）
- Lambda `weather_daily`（EventBridge毎日JST1:00に前日分を取得・保存）
- API Gateway `weather-api`（`GET /weather`、CORS設定済み）
- フロントエンドはAPI経由でDynamoDBから取得（Open-Meteo直叩きなし）

## AWS 構成（現時点）

```
フロントエンド（AWS Amplify）
  │
  ├── S3（bangumi-info）         ← 番組表CSV（毎週月曜更新）
  ├── S3（dodesca-video）        ← 動画（4/17のみ）
  └── API Gateway（weather-api）
        └── Lambda（weather_get）
              └── DynamoDB（weather_daily）
                    ↑
              Lambda（weather_daily）← EventBridge 毎日JST1:00
              Lambda（weather_batch_initial）← 初回のみ
                    ↑
              Open-Meteo API（名古屋 lat:35.18 lon:136.91）
```
