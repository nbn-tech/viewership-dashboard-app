# 視聴率ダッシュボード — Claude Code 指示ファイル

## 概要
名古屋地区在名7局の視聴率データをビジュアライズするReact SPA。
フレームワーク: React + Vite。実装は `src/App.jsx` の単一ファイル（約2500行）。

---

## デザインシステム

**必ず `DESIGN.md` を参照してから UI 変更を行うこと。**

DESIGN.md は Apple のウェブデザイン言語を分析したデザイン仕様書。
以下の主要トークンを必ず守る。

### カラー（`DESIGN.md` の colors セクション準拠）

| 用途 | トークン | 値 |
|---|---|---|
| **唯一のアクションカラー** | `primary` | `#0066cc` (Action Blue) |
| ページ背景（パーチメント） | `canvas-parchment` | `#f5f5f7` |
| ページ背景（白） | `canvas` | `#ffffff` |
| グローバルナビ背景 | `surface-black` | `#000000` |
| 本文テキスト | `ink` | `#1d1d1f` |
| 薄いテキスト | `ink-muted-48` | `#7a7a7a` |
| 区切り線 | `hairline` | `#e0e0e0` |
| ダーク面テキスト | `on-dark` | `#ffffff` |

### タイポグラフィ

- **フォントスタック**: `"SF Pro Display", "SF Pro Text", system-ui, -apple-system, BlinkMacSystemFont, sans-serif`
- **ボディ**: 17px / 400 / line-height 1.47 / letter-spacing -0.374px
- **ヘッドライン**: SF Pro Display / 600 / 負のレタースペーシング（-0.28〜-0.374px）
- **ナビリンク**: 12px / 400 / letter-spacing -0.12px
- **weight 500 は使用禁止**: 300 / 400 / 600 / 700 のみ

### 形状・丸み

| 用途 | 値 |
|---|---|
| プライマリCTA・ピル | `border-radius: 9999px` |
| ユーティリティカード | `18px` |
| コンパクトボタン | `8px` |

### ルール

**DO**
- インタラクティブ要素はすべて `#0066cc` を使用（リンク、ボタン、フォーカスリング）
- グローバルナビは `#000000` 背景、高さ44px
- ページ背景は `#f5f5f7`（パーチメント）または `#ffffff`
- ボタンのアクティブ状態は `transform: scale(0.95)`
- プライマリCTAは pill 形状（`border-radius: 9999px`）

**DON'T**
- 第2アクセントカラーを追加しない（`#0066cc` 一色のみ）
- ボタン・カード・テキストにグラデーション背景を使わない
- UI クロム（カード、ボタン、ナビ）にシャドウを付けない
- weight 500 は使用しない
- body テキストを16px以下にしない（17px が最小）

---

## アーキテクチャ

- スタイルはすべて JSX インラインスタイル（CSS モジュール・Tailwind なし）
- グローバルフォント・リセットは `src/index.css`
- デザイントークンは `DESIGN.md` 参照。新規 UI を追加する際は必ずトークンを確認
- 局ブランドカラーは `App.jsx` 先頭の `ST` 配列から参照（直接ハードコードしない）

## データ

- 視聴率実データ: `src/data/ratings_data.json`（名古屋地区・世帯視聴率、1分刻み）
- 番組表データ: S3 から日付ごとに取得（CSV または JSON）
- コーナー情報: `MORN_TPL` / `EVE_TPL` テンプレート＋POOLS（デモ用ダミーデータ）
- 4/17 ドデスカコーナー情報のみ実際の放送内容を分析したデータ
